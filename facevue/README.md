# facevue
A Vue-based adaptation of the "award-winning" application FACE, this repo pertains to go where no web application has hopefully gone before.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
